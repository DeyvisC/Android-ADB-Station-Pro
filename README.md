# Android ADB Station Pro 📱💻

A professional management tool built in **Python** to control, optimize, and secure Android devices (specifically tested on Redmi 9C) via **ADB**.

## 🛡️ Key Features
* **Privacy Shield:** One-click disable for Xiaomi/MIUI analytics and tracking daemons.
* **System Maintenance:** Deep cache cleaning and RAM optimization.
* **File Injector:** Automated media and music synchronization from PC to Device.
* **App Manager:** Advanced uninstaller for system bloatware.
* **Real-time Diagnostics:** Monitor RAM, Storage, and Battery temperature directly from the desktop.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **GUI:** CustomTkinter (Modern Dark Interface)
* **Backend:** ADB (Android Debug Bridge) via `subprocess`

## ⚠️ Requirements
1. **ADB Tools:** Must be installed on your PC.
2. **USB Debugging:** Must be enabled on the Android device.
3. **Python Libraries:** `pip install customtkinter`
